## IN-SCOPE (Allowed Relationships)

### 1. Light Precipitation Paradox:
- **light_precipitation + high_RH → aerosol_liquid_water → enhanced_chemistry → pm25_increase**
- **drizzle → prolonged_high_humidity → aqueous_reactions → sia_formation → pm25**
- **light_rain + stagnation → insufficient_scavenging + enhanced_moisture → pm25_increase**

### 2. Precipitation Intensity Thresholds:
- **precipitation_intensity → scavenging_efficiency → pm25_change** (threshold effects)
- **rainfall_rate < threshold → minimal_scavenging → pm25_persistence**
- **heavy_rain → efficient_scavenging → pm25_reduction**

### 3. Fog and Mist Effects:
- **fog + air_pollution → aerosol_activation → cloud_droplet_formation → aqueous_processing → sia_increase → pm25**
- **morning_fog → enhanced_RH → overnight_chemistry → pm25_buildup**
- **fog_persistence → reduced_visibility + enhanced_chemistry → pm25_increase**

### 4. Aerosol Liquid Water (ALW) Mechanisms:
- **high_humidity + aerosols → aerosol_liquid_water → aqueous_phase_reactions → secondary_aerosol_formation → pm25**
- **RH > 75% + light_precipitation → ALW_enhancement → sulfate_formation_acceleration → pm25**

### 5. Seasonal/Geographic Specific Cases:
- **tropical_precipitation_patterns → intermittent_rain → humidity_cycles → pm25_variations**
- **monsoon_onset_precipitation → high_humidity_persistence → enhanced_sia_formation → pm25**

### 6. Precipitation Chemistry Interactions:
- **acid_rain + NH3 → ammonium_formation_in_droplets → pm25_precursor_increase**
- **precipitation_pH + aerosol_precursors → in_cloud_processing → secondary_pm25_formation**

## OUT-OF-SCOPE (Exclude)

### Standard Wet Deposition:
- Simple rain → washout → PM2.5 decrease (already covered in other prompts)
- Standard scavenging coefficients without paradox aspects

### Emission Source Impacts:
- Traffic in rain, construction in rain (emission source category)
- Biomass burning affected by rain (emission source category)

### Pure Chemical Processes:
- Detailed aqueous chemistry reactions (refer to chemical_processes category)
- Cloud microphysics without PM2.5 impact
